WizAndroidBugMergePlugin
========================

为知笔记（Android版）Bug统计报告合并插件  
插件使用及编写注意：  

1. 配合为知笔记PC端使用  
2. PC端教程中本身要求js为Unicode，ini为Unicode，但git把Unicode作为二进制文件，不利于代码维护。  
本处保留js代码为UTF-8形式（需要提示时尽量使用英语，其他语言会导致PC乱码），ini为Unicode编码(防止识别乱码).
